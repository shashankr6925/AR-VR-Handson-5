
🌐 VR Physics Experience using Blender & Unity
📌 Overview

This project implements an interactive Virtual Reality (VR) experience using Blender for 3D modeling and Unity/A-Frame for simulation. The environment includes physics-based interactions where users can interact with objects in real-time.

The system demonstrates core concepts of VR such as immersion, object interaction, and real-time physics simulation.

🎯 Features

🧱 3D objects (cube, sphere) with physics properties

⚡ Real-time physics simulation (gravity, collisions)

🎮 Interactive objects (click to apply force)

👁️ VR-compatible scene using A-Frame

🌍 Simple environment with lighting and camera

🛠️ Technologies Used

Blender → For creating 3D models

Unity / A-Frame → For VR environment

JavaScript → For interaction logic

HTML & CSS → For structure and styling
